7th January 2016

PostgreSQL 9.5 Released!

The PostgreSQL Global Development Group is pleased to announce the availability of PostgreSQL 9.5

This is a new major release of PostgreSQL, including new features such as UPSERT, Row Level Security, BRIN indexes and other features for Big Data, performance improvements and much more.
