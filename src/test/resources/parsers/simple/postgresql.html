PostgreSQL 9.5 Alpha 2 Released!

6th August 2015

The PostgreSQL Global Development Group is pleased to announce the availability of PostgreSQL 9.5 Alpha 2!

This release contains previews of all of the features which will be available in the final release of version 9.5, although some details will change before then. Please download, test, and report what you find.
